Abstract
------------
사람들이 직접 경험한 오늘의 날씨정보를 토대로 옷차림을 공유한다면,   
내가 직접 느낀 날씨를 기록할 수 있다면 어떨까요?    
웨더피플은 날씨에 관련된 경험을 공유하고 , 소통하는 서비스 입니다.

이 레포지토리에는 백엔드 코드가 담겨있습니다.
